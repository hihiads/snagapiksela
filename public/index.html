O meni

Zovem se Din Mulajusufović, imam 15 godina i pohađam Tehničku školu za računalstvo i mrežne djelatnosti za smjer Tehničar za računalstvo. Programiranjem sam se počeo baviti iz znatiželje, a sada mi je to velika strast.

Iako sam početnik, već imam solidno iskustvo u razvoju web stranica koristeći HTML, CSS i JavaScript. Uživam učiti nove tehnologije i stalno radim na usavršavanju svojih vještina. Moj cilj je postati profesionalni web developer i stvarati moderne, responzivne i funkcionalne web aplikacije.

Vještine

  • HTML5
  • CSS3
  • JavaScript (osnovni projekti, DOM manipulacija, animacije)

Karakteristike

Odgovoran sam, organiziran i znatiželjan. Volim rješavati probleme, učiti nove stvari i raditi na projektima koji imaju stvarnu svrhu. Uvijek tražim prilike za suradnju i nove izazove.
